# Titanic-Classification-ML-Analysis
Titanic survival prediction using binary classification. Comparison of Logistic Regression, Gaussian Naive Bayes, and Decision Trees, including comprehensive EDA and model evaluation via ROC curves.  
This project focuses on developing a classification model to predict passenger survival on the Titanic, covering all essential stages from exploratory data analysis to comparative model evaluation. Performance was evaluated using two distinct approaches: Gaussian Naive Bayes, which achieved an accuracy of 79.81% and an AUC score of 0.85, demonstrating strong generalization and stability, and a Decision Tree model. Although the Decision Tree yielded slightly higher results on the test set (accuracy 80.75%, F1-score 74.53%), the analysis revealed a clear case of overfitting, as the model reached nearly 99% accuracy on the training data. By interpreting the confusion matrix and ROC curves, this project highlights the critical importance of model selection, concluding that GaussianNB remains more robust for unseen data, whereas the Decision Tree requires further pruning to prevent excessive complexity and overtraining.
